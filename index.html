CSS

  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element.
  • To group elements together, you can use a comma.
  • A single change to a CSS rule will apply to all of the tagged elements.
  • A selector defines the elements and attributes to which the rules and declarations apply.
  • Every HTML element is a box consisted of margins outside of the element. AKA CSS BOX MODEL
